# moat-texteffects
Cool text effects for Garry's Mod


## Glowing Text
__DrawGlowingText( static, text, font, x, y, color, xalign, yalign )__

static = Should the glow be constant instead of fading in and out?


## Fading Colors Text
__DrawFadingText( speed, text, font, x, y, color, fading_color, xalign, yalign )__

speed = speed at which the colors fade between each other
fading_color = the other color to fade to
